### 本项目将本公司的移动端官网改为vue版本

注：该项目在本地开发时通过抓取线上的API接口进行开发

### 技术栈：
vue2.0+vuex+axios+router(vue全家桶)+es2015(es6)

### 项目运行：
+ npm install
+ npm run dev
